# Discovery to Demo

A lightweight TypeScript web app that turns discovery notes into a demo narrative.

## Phases

- **Phase 0:** Runnable baseline with seeded discovery fields, local editing, and a concise generated output.
- **Phase 1:** Discovery readiness scoring, signal extraction, open questions, and a multi-step demo plan.

## Getting started

```bash
npm test
npm run build
npm run dev
```

Open <http://localhost:5173> after starting the development server.
